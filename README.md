`VITE_API_URL`

## Run Locally

Clone the project (Node >= 20.9.0)

```bash
  git clone https://github.com/vanthang24803/store-vue
```

Go to the project directory

```bash
  cd store-vue
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```

Website using port 3000

## Docker

Build Docker Image

```bash
  docker-compose up
```
